Пользовательское соглашение

Настоящее Пользовательское соглашение регулирует использование Telegram-бота Fasty и связанных с ним веб-сервисов. Используя Сервис, вы подтверждаете, что прочитали, поняли и согласны с условиями данного Соглашения.

1. Определения

Сервис — Telegram-бот Fasty (@ai_fasty_bot) и веб-редактор, доступный по адресу https://slidy-bot.vercel.app/, предоставляющие инструменты для генерации и редактирования контента.

Пользователь — физическое или юридическое лицо, использующее Сервис.

Поставщик услуг — Индивидуальный предприниматель Корешилов Егор Юрьевич, ИНН 352706623956, ОГРНИП 324350000002644, предоставляющий доступ к Сервису.

Контент — любые материалы (тексты, изображения, карусели), создаваемые с помощью Сервиса.

2. Описание сервиса

Fasty предоставляет пользователям следующие возможности:

  • Генерацию идей и текстов для контента с помощью технологий искусственного интеллекта (Claude API от Anthropic)
  • Создание структурированных постов и каруселей для социальных сетей
  • Подбор заголовков, описаний и хуков для постов и рилс
  • Настройку дизайна и экспорт готовых материалов

Сервис работает на базе следующих технологий:

  • Telegram Bot API для взаимодействия с пользователями
  • Claude API (Anthropic) для генерации контента
  • Продамус для приема платежей
  • Supabase для хранения данных

Сервис создан для упрощения процесса создания маркетингового контента и не гарантирует конкретных коммерческих результатов.

3. Регистрация и использование

Для использования требуется учетная запись Telegram. При первом запуске бота (@ai_fasty_bot) создается профиль пользователя.

Пользователь получает 3 бесплатные попытки для тестирования функционала.

Безлимитный доступ предоставляется по подписке стоимостью 1290 рублей в месяц с автоматическим продлением.

4. Условия подписки

Оплата производится через платежную систему Продамус. Подписка продлевается автоматически до момента отмены пользователем.

Возврат возможен в течение 14 дней после первого платежа, если Сервис не использовался (подробности в разделе "Политика возврата").

4.1. Управление подпиской

Отмена подписки:

Пользователь может отменить подписку в любой момент через Telegram-бота. Процесс отмены:

  1. Отправьте команду /subscription в боте @ai_fasty_bot
  2. Нажмите кнопку "Отменить подписку"
  3. Подтвердите отмену, нажав "Да, отменить"
  4. Бот отправит уведомление об успешной отмене

После отмены подписки:

  • Автоматические платежи прекращаются немедленно
  • Доступ к платным функциям сохраняется до конца текущего оплаченного периода
  • История использования и ранее созданный контент остаются доступны
  • После окончания периода доступно 3 бесплатные попытки

Альтернативный способ отмены:

  • Через личный кабинет платежной системы Продамус по ссылке из платежного письма
  • Обращением в поддержку @ekoreshilov с указанием Telegram ID

Приостановка подписки:

Функция временной приостановки в настоящее время не реализована в боте.

Для временного прекращения использования Сервиса необходимо отменить подписку через команду /subscription и при необходимости возобновить её позже.

Возобновление подписки:

После отмены подписку можно возобновить в любой момент:

  1. Отправьте команду /subscription в боте
  2. Нажмите кнопку "Оплатить подписку"
  3. Завершите оплату через Продамус
  4. Доступ активируется автоматически сразу после оплаты

При возобновлении подписки:

  • История использования сохраняется
  • Ранее созданный контент остается доступен
  • Подписка начинает действовать с момента оплаты
